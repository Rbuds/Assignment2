Binary Converter

This is a webpage I made with my knowledge of HTML and CSS that I learned in INFO 100 freshmen spring quarter with Katherine Diebel as my teacher. It's pretty basic and converts numbers to and from their 8-bit binary representation. The scripting approach is not as streamlined as it could be but works regardless and can accept numbers or binary sequences of any length. Give it a try here!

Sudoku

This webpage is one that I have been working on recently over the summer while flying to various locations. My original plan, given enough time, was to have this site be able to solve any difficulty puzzle. I was going to have the solve method there to act a sort of master copy to refer to when checking the clients inputs and to provide feedback as to correct or incorrect inputs. I ended up adding a button for the solve function anyways not only for debuggin but also just for fun and flash. I use a a series of strings, and text boxes to solve the puzzles. Although this is a very basic approach it works all the same. There are many things I would do to improve this further. I would make the solve formula recursive to eliminate excessive repetition. If you notice on the site you have to click the solve button three times to solve the easy puzzle. If recursion was added the button would have to be used once and would iterate through until the puzzle is finished. However given my limited knowledge of javascript I had to use some more primitive strategies. Warning: this project is still in beta testing phase. Algorithms are not complete and are not optimized.
